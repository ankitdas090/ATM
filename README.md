ATM System in C :

This is a simple ATM program written in C. 
It provides basic ATM functions like login, checking balance, depositing money, withdrawing money, and changing the PIN.
The program uses simple conditions, loops, and functions.

---

Features :

1. Login System

* User gets 3 attempts to enter the correct PIN.
* If correct, user can access the menu.
* If wrong 3 times, the card is blocked.

2. ATM Menu
   The menu has 5 options:
3. Check Balance
4. Deposit Money
5. Withdraw Money
6. Change PIN
7. Exit

---

Functional Details :

1. Check Balance
   Shows the available balance.

2. Deposit Money

* User enters an amount.
* Amount must be positive.
* Balance increases.

3. Withdraw Money

* User enters amount.
* Checks if valid and enough balance is available.

4. Change PIN

* User enters a new 4-digit PIN.
* Confirms the PIN again.
* Checks if it is valid.
